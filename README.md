# PD Structure Mastery (PWA)

A stylus-first (Apple Pencil / Android stylus) web app for Parkinson’s drug structure mastery:
- draw structures
- pass strict checkpoints (2 attempts + hint)
- tap-to-map required features
- export PNG/PDF
- offline-capable via service worker

## Deploy
Enable GitHub Pages: Settings → Pages → main branch → /(root)

## Install on tablet
- iPad Safari: Share → Add to Home Screen
- Android Chrome: Install app / Add to Home screen
